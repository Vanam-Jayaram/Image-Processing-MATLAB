# Image Processing
A MATLAB project that demonstrates fundamental image processing techniques using MATLAB's Image Processing Toolbox.
## Features
* Image Filtering
* Image Enhancement
* Edge Detection
* Digital Image Analysis
* MATLAB Image Processing Applications
## Technologies Used
* MATLAB
* Image Processing Toolbox
## Project Structure
* MATLAB source files (`.m`)
* Sample images (if included)
* Supporting documentation
## How to Run
1. Open MATLAB.
2. Open the project folder.
3. Run the main MATLAB script.
4. Select or load an image if required.
5. View the processed output.
